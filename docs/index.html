This page renders three animation technologies side by side — the same kind of motion the README shows off.

Lottie

lottie-web · boarding.lottie.json

Rive

@rive-app/canvas · live state machine

Animated SVG

pure SMIL · renders on GitHub too

WebGPU

navigator.gpu · WGSL shader, Canvas2D fallback

Note: the Quick Look preview itself never touches the GPU — a sandboxed preview extension stalls on Metal init, so the plugin renders barcodes on the CPU. WebGPU lives here, in the browser, only.

Install in 30 seconds

git clone https://github.com/ArioMoniri/ql-pkpass.git
cd ql-pkpass
make install        # builds, installs to /Applications, refreshes Quick Look

Then select examples/Skyline-BoardingPass.pkpass in Finder and press Space.
