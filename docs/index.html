🎫 Quick Look for Wallet passes

pkpass Quick Look is a native macOS plugin. Select any wallet pass in Finder, tap Space, and you get a real Wallet-style card — colours, logo, every field, and a scannable barcode. It reads Apple Wallet .pkpass, Google Wallet and Samsung Wallet passes. Everything renders on your Mac; nothing leaves it.

View on GitHub → Install

How it works

01

Select a pass

Click any pass file in Finder — or just hover it.

02

Glance or press Space

The preview pane shows the card; Space opens the full Quick Look with all details.

03

Read everything

Fields, barcode, the files inside, the raw JSON — and export to PDF from the app.

Works with every kind of pass

Not just flights — boarding passes, event tickets, coupons, loyalty and store cards, across all three wallets. Each card below is rendered by the actual plugin.

Install in 30 seconds

git clone https://github.com/ArioMoniri/ql-pkpass.git
cd ql-pkpass
make install        # builds, installs to /Applications, enables & refreshes Quick Look

Then select a pass in Finder and press Space. The helper lives in the menu bar — no Dock clutter.
